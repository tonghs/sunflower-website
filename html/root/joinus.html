产品经理

岗位职责：

1. 结合公司整体战略方向，根据市场调研情况，进行产品规划，负责公司互联网支付产品的设计与维护；

2. 负责 APP 和相关产品的工作，分析和挖掘用户需求，制定产品方案和进度，并执行与协调产品流程中的各个环节；

3. 负责移动互联网金融理财产品的热点和趋势分析，前期产品市场需求调研，产品创意和可行性分析，同类产品市场资料的收集；

4. 负责主导互联网金融移动平台产品的业务流程设计、功能设计、产品优化等工作；

5. 负责产品线的全面运营管理，保证用户量；

6. 对当前国内外金融市场、互联网金融等领域的创新产品进行研究与分析，设计完善现有的产品。

7. 跟踪上线产品的市场效果、用户反馈，收集和主动挖掘改进需求，根据业务需要持续改进产品，提升用户体验； 

8. 与营销、制作团队进行紧密配合，进行产品推广合作；

任职资格：

1. 金融、会计、财务、法律等相关专业本科以上学历；

2. 五年以上互联网产品开发经验，至少有一款成功产品的策划和运营经验，具有过往 P2P 线上产品、APP 客户端相关经验者优先；

3. 熟悉 B2B 网关支付、银企直连、银行卡协议支付、信用卡无卡支付等金融服务；

4. 对金融/理财领域有强烈兴趣，深刻理解股票、基金、理财产品等投资品种，对虚拟账户、银行结算等相关产品有经验者优先；

5. 熟练使用 OFFICE、AXURE 等工具软件，有投身互联网金融行业的强烈意愿和足够信心，有互联网金融行业移动产品经验优先；

6. 想象力丰富，有创意、有激情，良好的沟通能力和团队合作精神。

高级广告业务经理

岗位职责：

1. 负责知名品牌企业客户的开发与维护；

2. 电视节目广告销售与管理；

任职要求：

1. 五年以上媒体或广告行业销售工作经验；广告类、营销类等大专以上学历；

2. 较强的人际沟通、语言表达能力、适应及应变能力；

3. 精通销售技巧，有较强的客户开发意识，能独立开发客户；

4. 良好的团队协作精神，为人诚实可靠、品行端正、有亲和力；

5. 责任心强，服务意识强，能承受较强的工作压力；

6. 有媒体或 4A 广告公司工作经验、曾经服务过知名品牌客户者、有企业品牌客户资源者优先。

资深节目主编

岗位职责：

1. 负责节目内容运营，节目内容把控，对节目的内容品质和实效性负责；

2. 参与大型电视节目前期策划筹备，并跟进节目制作过程，对节目环节内容把关，对节目质量进行审核；

3. 负责项目推进及制作，以及项目其他相关工作的执行；

4. 对节目收视率和观众粘性负责，定期制定实施提高收视率的计划；

5. 负责节目团队的培训与评估，带领团队关注网络动态、影视热点，随时策划组织相关内容；

6. 在重要影视作品上线/开播前策划相关选题、制作专题；

7. 积极面对变化，带领团队不断进行内容运营创新，提升节目影响力；

8. 按节目规划落实节目的品牌推广、宣传发行以及后续市场开发的重要工作。

任职要求：

1. 影视节目制作专业本科以上学历，5 年以上省级卫视工作经验；

2. 丰富的大型活动、电视节目的策划制作经验，热爱影视/综艺类节目，有相当丰富的观影经验；

3. 关注并熟悉国内外影视产业动态，了解影视作品出品营销流程，能够从影视作品全生命周期上策划内容、运营内容；

4. 有丰富的互联网内容运营与团队管理、组织、培训经验；

5. 有良好的文字功底和策划创意能力；

6. 在内容运营方面有足够的敏感性，能够灵活应对外界变化策划相关选题；

7. 有很好的影像、图文、音乐鉴赏力，能够很好地把握各种内容形态和内容品质；

8. 责任心强，积极热情，有良好的协作精神，能承受一定的工作压力。

电视节目编导

岗位职责：

1. 有较好的理解能力、采编能力及文字功底；

2. 能熟练运用各种电视语言完成各类型节目或专题，具有对节目定位、策划、整体把控的能力；参与电视节目内容方面的策划、根据节目定位要求挖掘选题；负责节目前期的统筹工作；负责拍摄录制现场的调度与控制，协调整个制作过程；负责节目后期编辑剪辑、跟进后期制作，督促配合后期工作；协调沟通节目制作过程中的相关各环节，监控制作全程，保证节目质量；

3. 根据节目定位要求，负责组织编写节目策划案或剧本，拟定节目制作流程，把握节目风格，撰写脚本；

4. 会操作 Premiere、EDIUS、PHOTOSHOP 等基本节目制作软件；

5. 善于沟通，吃苦耐劳，抗压力强，积极乐观，能完成领导交予的工作；

6. 具备优秀的执行能力和协调能力以及良好的团队合作精神，具有敏锐的洞察力、良好的职业操守、较强的创新精神。

任职要求：

1. 新闻、编导类相关专业本科以上学历，熟悉电视节目制作流程；

2. 有独立编导作品，3 年以上省级卫视工作经验；

3. 有较强的新闻敏感性，有大型节目掌控能力；

4. 对节目的选题、编排、制作和对政策有掌控能力；

5. 具备文字撰写、话题策划、采访能力。
